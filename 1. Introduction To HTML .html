HTML - HTML (HyperText Markup Language) is the standard language used to create and design documents on the World Wide Web. The core structure of an HTML document consists of several essential tags, among which the <html>, <head> and <body> tags are fundamental.

< html > Tag
The < html > tag is the root element of an HTML document. It encapsulates all the content on the webpage, including the head and body sections.
Purpose: It informs the web browser that the document is an HTML document.

< head > Tag
The < head > tag contains metadata and links to external resources, such as stylesheets and scripts.
Purpose: It provides information about the document, such as its title, character set, styles, scripts, and other meta-information.

< body > Tag
The < body > tag contains all the contents of an HTML document, such as text, images, links, tables, lists, and other media.
Purpose : It represents the main content of the HTML document that is visible to the users.
