Tables in HTML

Tables in HTML are used to display tabular data in a structured format. A table is defined using the <table> tag, and it consists of rows (<tr>), headers (<th>), and data cells (<td>). Below, I'll provide an explanation of how to create tables in HTML, along with examples.

Basic Structure of an HTML Table

The basic elements of a table include:

  • <table> : Defines the table.
  • <tr> : Defines a row in the table.
  • <th> : Defines a header cell in the table.
  • <td> : Defines a standard data cell in the table.

Some attributes are used with <table> tag

  • Border: Can be added directly to the <table> tag to specify the width of the border around the table.
  • Padding: Padding is the space between the element's content and its border.
  • Spacing: Space between the individual cells of the table.
  • Margin: Margin is the space outside the table's border.
  • border-width: Sets the width of the border.
  • border-style: Sets the style of the border (e.g., solid, dashed, dotted, double, groove, ridge, inset, outset, none, hidden).
  • border-color: Sets the color of the border.
  • border-collapse: Specifies whether table borders are collapsed into a single border or separated.
