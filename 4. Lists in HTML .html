Lists

Lists in HTML are used to group a collection of related items in a specific order or sequence. HTML provides several types of lists, the most common being unordered lists, ordered lists, and description lists.

Types of lists in HTML :

Ordered Lists :

  1. Displays list items with numbers or letters.
  2. Implies a specific order or sequence for the items.
  3. Ordered lists start with <ol> tag and end with </ol>.
  4. Items are listed in ordered lists within <li> tags.
  5. We can also add attributes like 'type' (A, a, i, I, 1) and 'start' with the <ol> tag.

Example :

  1. Item 1
  2. Item 2
  3. Item 3

Unordered Lists :

  • Displays list items with bullet points.
  • Does not imply any particular order of the items.
  • Unordered lists start with <ul> tag and end with </ul>.
  • Items are listed in unordered lists within <li> tags.
  • We can also add attributes like 'type' (square, disc, circle) with the <ul> tag.

Example :

  • Item 1
  • Item 2
  • Item 3

Description Lists :

  • Used to create a list of terms and their descriptions.
  • Description lists start with <dl> tag and end with </dl>.
  • Consist of pairs of <dt> (definition term) and <dd> (definition description) elements.
